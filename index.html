⚙️ Configuración de Sincronización

Para activar la sincronización central con Google Sheets, sigue estos pasos:

  1. Abre script.google.com e inicia sesión con tu cuenta de Google
  2. Crea un nuevo proyecto y ponle nombre: Eurofarma Inventario
  3. Borra el código por defecto y pega el contenido del archivo apps_script.js
  4. Haz clic en Implementar → Nueva implementación
  5. Tipo: Aplicación web | Ejecutar como: Yo | Acceso: Cualquier persona
  6. Copia la URL de implementación que aparece
  7. Pégala abajo y haz clic en Guardar y probar
📋 Instrucciones para el equipo

Una vez configurada la URL del script:
• Comparte el archivo eurofarma_inventario.html con tu equipo via WhatsApp o Drive
• Cada representante abre el archivo en su navegador (Chrome o Edge recomendado)
• Ingresa la misma URL del script en su pestaña de Configuración
• Todos los registros se centralizan automáticamente en el Google Sheet
• Los datos se actualizan en tiempo real — sin instalar aplicaciones

💡 Tip: Puedes marcar la página como favorito en el celular para acceso rápido desde la pantalla de inicio.
🗓️ Gestión Mensual de Datos

Para mantener la data limpia cada mes, usa estas opciones. Solo el administrador debe ejecutarlas.

⚠️ Importante: Antes de limpiar, exporta el reporte CSV desde la pestaña 📊 Reportes para guardar el historial del mes.
🗑️ Limpiar mes específico

Elimina todos los registros de un mes y año determinado.

🚀 Nueva campaña mensual

Exporta el mes actual a CSV y luego limpia todos los registros para comenzar el nuevo mes.

⚠️ Limpiar TODO

Elimina absolutamente todos los registros del Sheet. Acción irreversible.
